A propos de moi

Je m'appelle Clément Le Bas et je suis développeur web en freelance, spécialisé sur le framework ReactJS. Je suis passionné par l'informatique et toujours prêt à découvrir de nouvelles technologies.
J'ai pu réaliser mon alternance chez 3V Finance ou j'ai développé une application web destiné aux entreprises qui souhaite gérer leurs risques financiers. Après 4 ans dans cette entreprise, j’ai décidé de mettre mon expérience à votre service pour vous accompagner dans la réussite de vos projets web.

Jobs

3V Finance

Développeur Front-end React • Septembre 2018 - Aujourd'hui
